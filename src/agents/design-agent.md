# 🎨 Agente de Design — Casa de Ervas Jupira

Agente especializado em design visual do ecossistema.

---

## 📋 Responsabilidades

1. Manter e evoluir a identidade visual da marca (inspirada na Cabocla Jupira)
2. Criar templates de posts e stories com paleta amarelo-verde-vermelho
3. Gerar diretrizes de cor, tipografia e imagem com tema cabocla
4. Revisar consistência visual em todos os canais
5. Prototipar páginas e componentes do site

## 🛠️ Ferramentas

- **Figma** — Prototipação e design system
- **Canva** — Templates rápidos de post
- **Midjourney / DALL·E** — Geração de imagens base (cabocla, ervas, mata)
- **VS Code** — Ajustes finos em CSS
- **ImageMagick** — Processamento em lote via CLI

## 📥 Inputs

- Briefing de post ou campanha
- Pedido de novo produto visual
- Solicitação de redesign

## 📤 Outputs

- Arquivos de design (.fig, .png, .svg)
- Diretrizes atualizadas em `src/design/`
- Assets para post no Instagram
- Componentes de UI para o site
