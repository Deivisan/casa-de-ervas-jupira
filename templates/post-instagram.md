## Template 1: Produto

```
┌──────────────────────┐
│                      │
│   ┌──────────────┐   │
│   │   IMAGEM DO   │   │
│   │   PRODUTO     │   │
│   │   (flat lay   │   │
│   │    amarelo +  │   │
│   │    verde)     │   │
│   └──────────────┘   │
│                      │
│   Nome do Produto    │
│   "Benefício em      │
│    uma frase"        │
│                      │
│   🔗 Link na bio     │
│                      │
└──────────────────────┘
```

**Legenda:**
```
✨ [NOME DO PRODUTO]

[Descrição sensorial — 2 a 3 frases sobre cheiro, textura, energia]

🌿 Como usar: [instrução rápida]

💚 [Benefício espiritual]

🙏 Okê Jupira!

🔗 Encontre na loja — link na bio

#casadeervasjupira #banhodeervas #ervasnaturais #[categoria] #okejupira
```
