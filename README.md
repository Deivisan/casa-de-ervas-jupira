# 🌿 Luigi de Aruanda

**Ecossistema criativo — posts, imagens, loja de ervas e design.**

Luigi de Aruanda é um projeto vivo que une espiritualidade, design e tecnologia. Aqui criamos:

- 📸 **Posts e stories para Instagram** com identidade visual mística
- 🎨 **Geração de imagens** com IA e curadoria artesanal
- 🌱 **Casa de Ervas** — loja online com banho, defumação, incenso e velas
- 🛒 **Site completo** com perfil visual, catálogo e checkout
- 🤖 **Agentes autônomos** para design, conteúdo, redes sociais e loja

---

## ⚡ Estrutura

```
├── agentes.md              # Orquestração dos agentes
├── src/
│   ├── design/             # Skill map, brand, cores, tipografia
│   ├── instagram/          # Calendário, templates, geração de imagens
│   ├── loja/               # Catálogo de ervas, site, checkout
│   └── agents/             # Agentes especializados (design, conteudo, social, loja)
├── assets/                 # Imagens, fontes, ícones
├── templates/              # Páginas e posts
└── config/                 # Workflows e automações
```

## 🚀 Status

Em construção ativa. Tudo será adicionado e reatualizado conforme o ecossistema cresce.
